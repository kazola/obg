# FLEAK

Python3 Flet + Bleak GUI console for Lowell Instruments' Bluetooth loggers.


## To install

```console
curl -L https://raw.githubusercontent.com/LowellInstruments/fleak/master/installer/installer_fleak.sh | bash
```
